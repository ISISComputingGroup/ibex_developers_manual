Stanford DG645 is a digital delay generator. There is an IOC, emulator and tests for it. The OPI is constructed in a modular fashion
out of smaller OPIs, to be reused if needed. The IOC is using a ['fork'](https://github.com/ISISComputingGroup/EPICS-delaygen) version of Delaygen, [original can be found here](https://github.com/epics-modules/delaygen)
