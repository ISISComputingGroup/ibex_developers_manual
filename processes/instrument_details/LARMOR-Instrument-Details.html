LARMOR

Warning

These instrument details pages provide historical context about the devices on a specific instrument, at the time they were migrated to IBEX. These pages may not be up to date with the latest instrument configuration.

This page collects information that will be useful for the implementation of the new control system on LARMOR.

Background & Timeline

The Larmor web page describes the background to the instrument. The layout of the LARMOR beamline is shown in this diagram. The proposed architecture for the LARMOR control system is described here. We also have a list of LARMOR’s motion control components.

Control System

Larmor is currently using the IBEX control system.
