CRISP

Warning

These instrument details pages provide historical context about the devices on a specific instrument, at the time they were migrated to IBEX. These pages may not be up to date with the latest instrument configuration.

This page collects information that will be useful for the implementation of the IBEX control system on CRISP.

Background & Timeline

CRISP is a reflectometer instrument at ISIS, on TS1. The CRISP web page describes the background to the instrument.

Note: DAE

See multi-detector and single-detector below.

Note: Choppers

CRISP has a Mk3 chopper.

Note: Jaws

Provide information about CRISP jaws.

Note: Vacuum System

TPG300 support is implemented via #216 and #2063

Note: Eurotherm

Eurotherms are used to control temperature Orange Cryostat, CCR and Furnace devices.

Note: Oxford Instruments

  1. Orange Cryostat: models to be determined. Orange cryostats are controlled via a temperature controller (e.g. Eurotherm), so nothing on the cryostat itself for IBEX to control.

  2. Blue Cryostat: models to be determined.

Note: Closed Cycle Refrigerators

  1. CCR is the most frequently used low-temperature device used in on CRISP. CCRs. Not directly computer-controlled - controlled via Eurotherm.

Note: Water Baths

CRISP uses two types of water bath

  1. Julabo (which model(s)?)

  2. Grant Water Bath

    • Update (25-06-2019): Haake water baths are no longer used. No longer any need to support them. See #4456

Note: Keithley

  1. Keithley 2000 Series Multi-Meter. See also #2460.

  2. Keithley 2600 Series Source Meter.

Note: Hirst

Hirst GM05 gaussmeter

Note: Hitachi

Hitachi pumps are no longer supported in ISIS

Note: Knauer

Knauer K-6 Electric Valve Drive. Knauer web site
K-6 model appears to have been superseded.

Note: Knauer HPLC

Knauer HPLC 1050 Knauer HPLC 1050 is discontinued..

Note: Nima Trough

Nima Trough: SECI used a manufacturer supplied VI. We may need to do the same in IBEX (via lvDCOM).

  • Note: NIMA Technologies Ltd now seems to be part of Biolin Scientific.

  • The NIMA trough is used regularly on SURF. The manufacturer supplied VI is used to view graphs showing information about thin films.

  • The manufacturer has made additional software available for download. A copy of this software is located in \\isis\shares\ISIS_Experiment_Controls\NIMA Trough\Nima_TR8.1.zip.

Note: Thurlby

Thurlby EX355P PSU - see #155 and #198.

Thurlby Thandar Instruments EX355P PSU

Note: Syringe Pumps

  1. Watson Marlow 323 Peristaltic Pump

  2. WPI Aladdin-1000 Syringe Pump

  3. WPI SP2xx Syringe Pump - check specific model.

Note: He3 Monitor

He3 Monitor is an obsolete piece of equipment. It is no longer used. No need to support it.
