Data streaming how-to guide

This is a guide for basic operations using either the development or production Kafka clusters we use for data streaming at ISIS.

Note that there are many ways to do the following, what is written here is the way commonly done at ISIS on our development and production clusters. Something like kafka-tool is a nice GUI that will list topics, brokers, etc and create or delete topics. You may have more luck running things like kafkacat, kafkacow or any of the official Kafka scripts under the Windows subsystem for linux
