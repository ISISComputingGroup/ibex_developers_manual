Moxa NPort

To install/upgrade NPort driver:

  1. Use remote desktop because you need admin privileges

  2. Open NPort admin and note the IP address, Com ports and moxa ports for the current config (NB there maybe be multiple moxas)

  3. Uninstall NPort using windows add/remove programs

  4. Install the latest NPort software from ...\installs\Installs\Applications\LabVIEW\Other bits\LabVIEW related\MOXA\MOXA NPort

  5. Open NPort Admin

  6. Click Add

  7. Search for the NPort (click stop when it times out)

  8. Clear All, then select the moxa you are using

  9. Click OK, do not activate ports

  10. Select all ports

  11. Click Settings

  12. Under basic settings Click “Auto Enumerating COM Number for Selected Ports”

  13. Set Com to correct port (probably port 5)

  14. Click OK.

  15. Click Apply.

  16. Ignore the message about setting moxa to real port.

  17. Close NPort administrator

If the upon clicking apply you are told the port is in use, you may need to move to using NPort Driver Manager instead, it can be found here \\isis\inst$\Kits$\CompGroup\Utilities\MOXA Nport Software
