Test Machines

Test machines are currently virtual machines running on … to access them:

  1. Run Hyper-V Manager using your admin account (you may have to install it)

  2. Connect to the server ndhspare70 (use right click, I think I used my admin credentials and Chris had to enable access for me)

To reset them

  1. Click on the server (left)

  2. Click the Virtual machine you want to reset (middle)

  3. Right click on the Clean for Clone check point and click apply ...

  4. Click Apply yes I do

  5. Right click and delete checkpoint sub tree (above clean for clone) - this removes history

  6. Bring up the machine console (double click on the machine name)

  7. Click power on (green button left on toolbar)

  8. Wait for boot

  9. Task Sequence: Build Basic W7

  10. Computer Details: Set machine name and next

  11. Move Data and Settings: Do not move

  12. User Data: Do not restore

  13. Administration Password: Leave as is

  14. Ready: Begin

  15. Leave to boot
